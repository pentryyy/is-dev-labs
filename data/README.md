# Описание датасета: Прогноз сердечно-сосудистых заболеваний

## Общая информация

Датасет предназначен для решения задачи бинарной классификации — предсказания наличия или отсутствия сердечно-сосудистых заболеваний (ССЗ) у пациентов на основе медицинских и демографических показателей.

## Описание датасета

Датасет содержит медицинские показатели пациентов для задачи бинарной классификации наличия сердечно-сосудистых заболеваний.

| Признак | Описание | Тип | Пример значений |
| :--- | :--- | :--- | :--- |
| `id` | Уникальный идентификатор пациента | integer | 0, 1, 2... |
| `age` | Возраст пациента в днях | integer | 14507–23589 (~40–65 лет) |
| `gender` | Пол | categorical | 1, 2 *(требуется уточнение кодировки)* |
| `height` | Рост | float (см) | 148–185 |
| `weight` | Вес | float (кг) | 45.0–115.0 |
| `ap_hi` | Систолическое («верхнее») давление | integer (мм рт. ст.) | 90–200 |
| `ap_lo` | Диастолическое («нижнее») давление | integer (мм рт. ст.) | 60–110 |
| `cholesterol` | Уровень холестерина | ordinal (1–3) | 1 = норма, 2 = выше нормы, 3 = значительно выше |
| `gluc` | Уровень глюкозы | ordinal (1–3) | 1 = норма, 2 = выше нормы, 3 = значительно выше |
| `smoke` | Курение | binary | 0 = нет, 1 = да |
| `alco` | Употребление алкоголя | binary | 0 = нет, 1 = да |
| `active` | Физическая активность | binary | 0 = низкая, 1 = высокая |
| `cardio` | **Целевая переменная**: наличие ССЗ | binary | 0 = нет, 1 = есть |
